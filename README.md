# 📚 AI Study Planner System

An intelligent **AI-powered Study Planner System** designed to help students organize study schedules, manage tasks, track progress, and improve productivity through personalized AI recommendations and interactive data visualization dashboards.

The system analyzes study habits, performance trends, strengths, and weaknesses to generate optimized study plans and improve learning efficiency.

## 🚀 Features

- 🧠 AI-generated personalized study plans
- 📅 Smart scheduling and study task management
- ⏰ Intelligent reminders and notifications
- 📊 Interactive dashboards with charts and analytics
- 📈 Subject-wise progress tracking
- 🎯 Goal setting and exam preparation support
- 🔍 AI recommendations based on performance analysis
- 📉 Productivity monitoring and study pattern analysis
- ✅ Task completion tracking
- 💻 Responsive and user-friendly interface

## ⚙️ Installation & Setup

### Step 1: Clone the repository

```bash
git clone https://github.com/your-username/AI-Study-Planner-System.git
```

### Step 2: Move to project folder

```bash
cd AI-Study-Planner-System
```

### Step 3: Install required dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run the application

```bash
python app.py
```

### Step 5: Open in browser

```bash
http://127.0.0.1:5000/
```
